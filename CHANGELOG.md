# Changelog

## Version 1.0

Initial stable release.

- Added Auto-Forward Smart Speed keybind.
- Added automatic initial speed detection.
- Added jogging start when activated from idle.
- Added current speed preservation when activated while already moving.
- Added speed adjustment while auto-forward is active, using the player’s configured walk and sprint controls.
- Added clean stop conditions using the auto-forward key, movement keys, forward/backward input, jump and vault.
- Added Character category sorting for the keybind.
- Reworked internal prefix and configuration under AFSS.
- Added signed PBO and public key support.
