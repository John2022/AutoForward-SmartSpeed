# Contributing / Contribution

## English

Contributions are welcome if they respect the licence and the credit requirements.

Before submitting a pull request:

- keep the credits to John2022FR and GayCookie / QuickStep;
- clearly describe what changed;
- test the mod in game;
- do not include private keys;
- do not change the licence without written permission from John2022FR;
- do not remove the free access requirement;
- do not claim original authorship of Auto-Forward Smart Speed or QuickStep.

Accepted contributions may include:

- bug fixes;
- compatibility fixes;
- documentation improvements;
- translation improvements;
- code cleanup that does not change the validated behavior;
- optional improvements if they keep the mod lightweight and clear.

## Français

Les contributions sont acceptées si elles respectent la licence et les obligations de crédit.

Avant de soumettre une pull request :

- conservez les crédits de John2022FR et GayCookie / QuickStep ;
- décrivez clairement ce qui a été modifié ;
- testez le mod en jeu ;
- n’incluez aucune clé privée ;
- ne modifiez pas la licence sans autorisation écrite de John2022FR ;
- ne supprimez pas l’obligation d’accès gratuit ;
- ne revendiquez pas la paternité originale d’Auto-Forward Smart Speed ou de QuickStep.

Les contributions acceptables peuvent porter sur :

- des corrections de bugs ;
- des corrections de compatibilité ;
- des améliorations de documentation ;
- des améliorations de traduction ;
- du nettoyage de code qui ne modifie pas le comportement validé ;
- des améliorations optionnelles si elles conservent un mod léger et clair.
