# Auto-Forward Smart Speed

![Auto-Forward Smart Speed preview](Workshop/@AutoForward-SmartSpeed/afss_preview.png)

Auto-Forward Smart Speed adds a smart auto-forward keybind for DayZ. It starts from your current movement state and lets you adjust speed using your configured walk and sprint controls (Ctrl & Shift by default).

Auto-Forward Smart Speed ajoute une commande d’avance automatique intelligente pour DayZ. Le mod démarre selon votre état de déplacement actuel et permet d’ajuster la vitesse avec vos commandes configurées de marche et de sprint (Ctrl & Maj par défaut).

Steam Workshop: https://steamcommunity.com/sharedfiles/filedetails/?id=3746092811

## English

### Description

Auto-Forward Smart Speed adds a single smart auto-forward keybind to DayZ.

When enabled, the mod automatically keeps your character moving forward. It is designed to make long movements more comfortable while keeping manual control responsive and easy to recover at any time.

### Features

- One dedicated keybind: Auto-Forward Smart Speed.
- Starts in jogging mode when activated while standing still.
- Keeps the current movement speed when activated while already moving.
- Supports walking, jogging and running.
- Allows speed adjustment while auto-forward is active.
- Uses the player’s configured walk and sprint controls, not hardcoded keys.
- Stops cleanly when using the auto-forward key again, movement keys, forward/backward input, jump or vault.

### Technical information

- Mod folder: `@AutoForward-SmartSpeed`
- PBO: `AFSS.pbo`
- Prefix: `AFSS`
- Input action: `UAAFSSAutoForward`
- Version: `1.0`
- Dependencies: none

## Français

### Description

Auto-Forward Smart Speed ajoute une commande unique d’avance automatique intelligente à DayZ.

Une fois activé, le mod maintient automatiquement le déplacement vers l’avant de votre personnage. Il est conçu pour rendre les longs déplacements plus confortables, tout en conservant un contrôle manuel réactif et facile à reprendre à tout moment.

### Fonctionnalités principales

- Une seule commande dédiée : Auto-Forward Smart Speed.
- Démarrage en jogging lorsque le mod est activé à l’arrêt.
- Conservation de la vitesse actuelle lorsque le mod est activé en mouvement.
- Prise en charge de la marche, du jogging et de la course.
- Ajustement de la vitesse pendant que l’avance automatique est active.
- Utilisation des commandes configurées de marche et de sprint du joueur, sans touches codées en dur.
- Arrêt propre par la touche Auto-Forward, les touches de déplacement, avancer/reculer, saut ou franchissement.
