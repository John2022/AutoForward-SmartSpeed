# Security Policy / Politique de sécurité

## English

Do not report normal gameplay bugs through private security channels.

Open a public issue for:

- keybind issues;
- installation issues;
- server loading issues;
- signature mismatch;
- mod conflicts;
- auto-forward behavior issues.

Report privately only if the issue exposes sensitive information, private keys, credentials, or a real security risk outside normal gameplay.

This repository must never contain `John2022FR_DayZMods.biprivatekey` or any other private signing key.

---

## Français

Ne signalez pas les bugs de gameplay ordinaires par un canal de sécurité privé.

Ouvrez une issue publique pour :

- les problèmes de keybind ;
- les problèmes d’installation ;
- les problèmes de chargement serveur ;
- les erreurs de signature ;
- les conflits avec d’autres mods ;
- les problèmes de comportement de l’avance automatique.

Un signalement privé ne doit être utilisé que si le problème expose des informations sensibles, des clés privées, des identifiants ou un vrai risque de sécurité hors gameplay normal.

Ce dépôt ne doit jamais contenir `John2022FR_DayZMods.biprivatekey` ni aucune autre clé privée de signature.
