# Support / Assistance

## English

Before opening an issue, please check:

- [Installation](docs/INSTALLATION.md)
- [Troubleshooting](docs/TROUBLESHOOTING.md)
- [Server owners notes](docs/SERVER_OWNERS.md)

For bugs, use the Bug Report issue form.

For suggestions, use the Feature Request issue form.

For server installation or keybind issues, use the Server / Setup Help issue form.

---

## Français

Avant d’ouvrir une issue, consultez :

- [Installation](docs/INSTALLATION.md)
- [Dépannage](docs/TROUBLESHOOTING.md)
- [Notes pour administrateurs de serveurs](docs/SERVER_OWNERS.md)

Pour un bug, utilisez le formulaire Bug Report.

Pour une suggestion, utilisez le formulaire Feature Request.

Pour un problème d’installation serveur ou de commande, utilisez le formulaire Server / Setup Help.
