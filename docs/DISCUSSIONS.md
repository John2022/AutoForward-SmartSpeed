# GitHub Discussions / Discussions GitHub

## English

GitHub Discussions cannot be fully enabled by files alone. They must be enabled in the repository settings.

Suggested setup:

1. Open the GitHub repository.
2. Go to `Settings`.
3. Go to `Features`.
4. Enable `Discussions`.
5. Create or keep categories such as:
   - General
   - Questions
   - Ideas / Suggestions
   - Server owners
   - Repack / derivatives

Suggested first discussion:

Title:
Welcome to Auto-Forward Smart Speed discussions

Body:
Use this space for general questions, server owner feedback, suggestions and compatibility discussions. For reproducible bugs, please use GitHub Issues instead.

---

## Français

Les Discussions GitHub ne peuvent pas être entièrement activées par de simples fichiers. Elles doivent être activées dans les paramètres du dépôt.

Configuration conseillée :

1. Ouvrir le dépôt GitHub.
2. Aller dans `Settings`.
3. Aller dans `Features`.
4. Activer `Discussions`.
5. Créer ou conserver des catégories comme :
   - General
   - Questions
   - Ideas / Suggestions
   - Server owners
   - Repack / derivatives

Première discussion conseillée :

Titre :
Welcome to Auto-Forward Smart Speed discussions

Contenu :
Utilisez cet espace pour les questions générales, les retours d’administrateurs de serveurs, les suggestions et les discussions de compatibilité. Pour les bugs reproductibles, utilisez plutôt les GitHub Issues.
