# Repack and derivatives / Repack et versions dérivées

## English

Repacking, modifying and redistributing this mod is allowed under the licence conditions.

Credits must remain visible. Modified versions must be clearly identified as modified versions. Access to the mod itself must remain free.

## Français

Le repack, la modification et la redistribution de ce mod sont autorisés dans les conditions prévues par la licence.

Les crédits doivent rester visibles. Les versions modifiées doivent être clairement identifiées comme telles. L’accès au mod lui-même doit rester gratuit.
