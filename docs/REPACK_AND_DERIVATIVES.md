# Repack, redistribution and derivatives / Repack, redistribution et versions dérivées

## English

Repack and redistribution are allowed, but only under the licence conditions.

You may:

- use the mod on your server;
- include the mod in a server pack;
- modify the mod for your own needs;
- publish a modified version;
- redistribute the mod elsewhere.

You must:

- credit John2022FR for Auto-Forward Smart Speed;
- credit GayCookie for QuickStep, the original technical base;
- clearly identify modified versions as modified versions;
- keep access to the mod itself free;
- keep the credit history visible.

You may not:

- claim original authorship;
- remove or hide credits;
- slightly modify the code and present the result as an original work;
- sell access to the mod itself without written permission;
- include the mod in a paid modpack without written permission.
