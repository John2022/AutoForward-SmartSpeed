# Server owners / Administrateurs de serveurs

## English

### Monetized servers

Monetized DayZ servers may use this mod.

The rule is simple:

- monetized server using the mod: allowed;
- donations, VIP systems, priority queue payments or cosmetic payments: allowed;
- free mod required to join or play on that server: allowed;
- paid access to the mod itself: not allowed without written permission;
- selling the mod as part of a paid modpack: not allowed without written permission.

The mod itself must remain freely accessible to players.

If access to the mod itself is monetized, written permission from John2022FR is required and may involve a royalty agreement.

### Repack and redistribution

Repack and redistribution are allowed under the licence conditions.

You must keep clear credits to:

- John2022FR for Auto-Forward Smart Speed;
- GayCookie for QuickStep, the original technical base.

You must not present a repacked or modified version as your original work.

### Server loading

If the mod is installed in the server root folder, the server `-mod=` line can include:

`@AutoForward-SmartSpeed`

If you use absolute paths, point to the folder containing `Addons`, `keys`, `mod.cpp`, `meta.cpp` and `afss_preview.png`.

## Français

### Serveurs monétisés

Les serveurs DayZ monétisés peuvent utiliser ce mod.

La règle est simple :

- serveur monétisé utilisant le mod : autorisé ;
- dons, systèmes VIP, files prioritaires ou paiements cosmétiques : autorisés ;
- mod gratuit requis pour rejoindre ou jouer sur ce serveur : autorisé ;
- accès payant au mod lui-même : interdit sans autorisation écrite ;
- vente du mod dans un modpack payant : interdite sans autorisation écrite.

Le mod lui-même doit rester accessible gratuitement aux joueurs.

Si l’accès au mod lui-même est monétisé, une autorisation écrite de John2022FR est nécessaire et peut impliquer un accord de royalties.

### Repack et redistribution

Le repack et la redistribution sont autorisés dans le respect des conditions de licence.

Les crédits doivent rester clairement visibles :

- John2022FR pour Auto-Forward Smart Speed ;
- GayCookie pour QuickStep, la base technique originale.

Vous ne devez pas présenter une version repackée ou modifiée comme votre œuvre originale.

### Chargement serveur

Si le mod est installé dans le dossier racine du serveur, la ligne `-mod=` peut inclure :

`@AutoForward-SmartSpeed`

Si vous utilisez des chemins absolus, pointez vers le dossier contenant `Addons`, `keys`, `mod.cpp`, `meta.cpp` et `afss_preview.png`.
