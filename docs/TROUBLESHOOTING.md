# Troubleshooting / Dépannage

## English

Common checks:

- Make sure the mod is loaded on the client.
- Make sure the server has the matching signed PBO.
- Make sure the server has the public key.
- Do not load both a local version and the Workshop version at the same time.
- Check the DayZ RPT logs if the game fails to start.

## Français

Vérifications courantes :

- Vérifiez que le mod est chargé côté client.
- Vérifiez que le serveur utilise le PBO signé correspondant.
- Vérifiez que le serveur possède la clé publique.
- Ne chargez pas en même temps une version locale et la version Workshop.
- Consultez les logs RPT de DayZ si le jeu ne démarre pas.
