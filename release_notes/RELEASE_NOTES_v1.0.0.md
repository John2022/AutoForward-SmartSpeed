# Auto-Forward Smart Speed v1.0.0

## English

Initial stable release.

### Added

- Auto-Forward Smart Speed keybind.
- Automatic initial speed detection.
- Jogging start when activated from idle.
- Current speed preservation when activated while already moving.
- Speed adjustment while auto-forward is active, using the player’s configured walk and sprint controls.
- Clean stop conditions using the auto-forward key, movement keys, forward/backward input, jump and vault.
- Character category sorting for the keybind.
- Signed PBO and public key support.

## Français

Première version stable.

### Ajouts

- Commande Auto-Forward Smart Speed.
- Détection automatique de la vitesse initiale.
- Démarrage en jogging lorsque le mod est activé à l’arrêt.
- Conservation de la vitesse actuelle lorsque le mod est activé en mouvement.
- Ajustement de la vitesse pendant que l’avance automatique est active, avec les commandes configurées de marche et de sprint du joueur.
- Conditions d’arrêt propre avec la touche Auto-Forward, les touches de déplacement, avancer/reculer, saut et franchissement.
- Rangement de la commande dans la catégorie Personnage.
- PBO signé et clé publique.
